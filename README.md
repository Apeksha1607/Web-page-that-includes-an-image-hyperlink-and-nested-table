# 🎓 Student Digital Profile Card

A simple **HTML + CSS project** that displays a **student learning hub card** with a profile icon, description, learning resource link, and styled table.

This project is beginner-friendly and helps practice **basic web design concepts** like layout, styling, and responsive card design.

---

## 📌 Features

* Clean **card UI design**
* **Gradient background**
* Profile icon with circular styling
* External learning link
* Styled **HTML table**
* Simple and beginner-friendly code

---

## 🛠 Technologies Used

* **HTML5**
* **CSS3**

---

## 📷 Preview

The page contains:

* Student learning card
* Profile icon
* Link to W3Schools for learning web development
* Table describing the project purpose
